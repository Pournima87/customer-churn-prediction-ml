# Customer Churn Prediction using Machine Learning

## Project Overview
Customer churn is one of the biggest challenges for telecom companies.  
This project builds a supervised machine learning model to predict which customers are likely to leave and identifies key factors contributing to churn.

The goal is not only to predict churn but also to extract actionable insights that help businesses reduce customer loss.

---

## Dataset
Telco Customer Churn Dataset

- 7032 customer records
- 21 features
- Target variable: Churn (Yes/No)

Features include:
- Customer tenure
- Contract type
- Internet service
- Payment method
- Monthly and total charges

---

## Key Project Steps

1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Engineering
4. Model Training
5. Model Comparison
6. Cross Validation
7. Model Interpretation
8. Risk Segmentation

---

## Models Evaluated

- Logistic Regression
- Decision Tree
- Random Forest
- Gradient Boosting
- KNN
- SVM

Best performing model:

**Logistic Regression**

Performance:

- ROC-AUC: **0.8458**
- 5-fold cross-validation
- Stable generalization

---

## Key Insights

Major churn drivers:

- Month-to-month contracts
- Fiber optic service
- Electronic check payments
- Short customer tenure

Protective factors:

- Long-term contracts
- Tech support
- Online security services

---

## Risk Segmentation

Customers were categorized into:

- Low Risk
- Medium Risk
- High Risk

Results:

- High-risk segment: **14% customers**
- **69% of them actually churned**

This allows businesses to target retention strategies effectively.
